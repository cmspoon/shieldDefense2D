================================================================================
                              SHIELD DEFENSE 2D
================================================================================

DESCRIPTION
----------- 
You must defend the central power core from the enemies appproaching from both
sides. The power core is protected by a shield, only strong enough to withstand
four hits from the enemies. If the enemies break the shield and destory the
power core, the game is OVER!

CONTROLS
----------------------
[ Keyboard ]
A / D              : Move the player left and right
SPACE              : Jump
Mouse              : Aim the gun
LMB (Hold)         : Fire bullets toward the cursor's location
+                  : Zoom the camera IN
-                  : Zoom the camera OUT
ESC                : Close the game
